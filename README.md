# 🏪 Smart CPG Inventory & Sales Management System

A comprehensive enterprise-grade inventory management system for Consumer Packaged Goods (CPG) companies with AI-powered forecasting and role-based access control.

## 🎯 Features

- **📊 Real-Time Dashboard** - Interactive analytics and visualizations
- **📦 Inventory Management** - CRUD operations with low-stock alerts
- **💰 Sales Tracking** - Multi-region sales recording and analysis
- **🤖 AI Forecasting** - ML-powered sales prediction using XGBoost
- **🔐 Role-Based Access** - Admin, Manager, and Employee roles
- **📧 Email Alerts** - Automated low-stock notifications
- **📈 Analytics & Reports** - Comprehensive business intelligence
- **📥 Data Export** - Excel and CSV export functionality

## 🛠️ Technology Stack

- **Frontend**: Streamlit, Custom CSS/HTML
- **Backend**: Python, SQLite
- **Data Analysis**: Pandas, NumPy
- **Visualization**: Plotly
- **Machine Learning**: XGBoost, scikit-learn
- **Authentication**: streamlit-authenticator, bcrypt
